# MediTrack

A comprehensive pharmacy management system built with Electron, React, and TypeScript.

## Features

- Inventory Management
- Sales Tracking
- Alerts & Notifications
- Analytics & Reporting
- User Management
- Data Import/Export
- Backup & Restore

## Project Structure

```
meditrack-app/
├── desktop/          # Electron + React desktop app
├── backend/          # Express.js backend (optional)
├── shared/           # Shared utilities
└── docs/             # Documentation
```

## Getting Started

### Desktop App

```bash
cd desktop
npm install
npm run dev
```

### Build

```bash
npm run build
```
